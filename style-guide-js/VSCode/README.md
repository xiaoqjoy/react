1. ESLint

## Settings.json 配置

```json
{
    "editor.fontSize": 16,
    "editor.formatOnSave": true,
    "prettier.semi": true,
    "prettier.singleQuote": true,
    "eslint.autoFixOnSave": true,
    "prettier.tabWidth": 4,
    "prettier.jsxSingleQuote": true
}
```
